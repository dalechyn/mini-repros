### 👷🏻‍♂️ compile-wagmi-cli-lib

### ⬆️ Installation
```sh
npm i
```

### 🚀 Target
Compile the generated `wagmi.hooks.ts` with `declaration: true` to be able to
use the generated hooks along with declaration files in any other project.

### 🪄 Build
```sh
npm run build
```
